1. Hexo 简介

Hexo 是一个基于 Node.js 的静态博客框架，通过 Markdown 编写内容并生成静态网页。它具有以下特点：

  • 轻量快速：生成静态页面速度快，适合快速搭建个人博客。
  • Markdown 支持：使用 Markdown 语言编写内容，简单且易于阅读。
  • 插件丰富：提供了多种插件和主题，方便扩展功能和美化博客。
  • 部署简便：生成的静态页面可以直接部署到 GitHub Pages、Netlify 等服务。
